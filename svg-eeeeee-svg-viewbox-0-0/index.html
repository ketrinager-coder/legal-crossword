Интерактивный кроссворд для проверки знаний по уголовно-исполнительному праву

Выберите вопрос, введите термин и проверьте правильность ответа. Кроссворд помогает повторить ключевые понятия темы в формате учебной игры.

Как играть
  • Выберите вопрос по горизонтали или вертикали.
  • Прочитайте определение и введите ответ в поле.
  • Нажмите «Проверить» для проверки активного термина.
  • Правильные ответы подсвечиваются зелёным.
  • Ошибки подсвечиваются красным и не засчитываются.
  • Прогресс отображается в верхней панели.
